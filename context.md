- **카테고리 탭 (Tab) 내비게이션 도입**: 기존에 세로로 나열되던 포트폴리오 섹션(`메인 프로젝트`, `게임 분석`, `AI 활용`)을 상단 탭 형식으로 변경하여 유저가 원하는 카테고리만 직관적으로 필터링하여 볼 수 있도록 개선. (기타 작업물 탭 삭제)
- **화이트톤 카드 레이아웃 적용**: 어두운 유리 질감의 카드 디자인을 화이트 배경(`bg-white`), 부드러운 테두리(`border-[#DBE2EF]`), 넓은 이미지 영역(`aspect-[4/3]`)으로 전면 교체하여 깔끔하고 모던한 아카이브 형태로 개선.
- **클릭 범위 및 상세 뷰 동기화**: `상세보기` 버튼을 클릭해야만 이동하던 방식에서, 카드 전체 영역을 클릭해도 프로젝트 상세 페이지로 즉시 이동할 수 있도록 접근성 향상.
- **플랫폼 링크 태그 고도화**: 일반 태그(회색 테두리)와 링크 이동이 가능한 특수 플랫폼 태그(Steam, Google Play 등)를 디자인적으로 명확히 분리. 관리자 모드에서 해당 태그들에 대한 URL 연결을 동적으로 편집할 수 있는 기능 추가.
- **설명 문구 동적 편집 기능 추가**: 포트폴리오 갤러리 상단 설명 문구를 하드코딩에서 `portfolio_description` Supabase 상태로 분리하여 관리자 모드에서 텍스트 에디터로 즉시 편집 및 저장되도록 개선.
- **카테고리 탭 동적 편집 연동**: 포트폴리오 갤러리의 카테고리 탭(게임 분석, AI 활용 등) 텍스트를 관리자가 직접 수정할 수 있도록 `portfolio_categories` 상태를 추가하고, 카테고리명 변경 시 해당 카테고리에 속한 모든 프로젝트의 데이터도 자동 업데이트되도록 구현.
- **게임 출시 전용 플랫폼 태그 신규 기능**: 포트폴리오 카드 상세 설명 아래 구분선 하단에, '게임 출시'를 나타낼 수 있는 커스텀 태그(`releaseTags`) 영역을 새로 추가. 이미지 아이콘 업로드, 라벨 텍스트 지정, 외부 링크 연결 기능 지원. 일반 모드에서 등록된 태그가 없을 경우 영역 전체가 숨겨짐. (기존 데이터 구조 유지)
- **일반 태그 자동 분리 기능 제거**: 포트폴리오 프로젝트의 위쪽 일반 태그에서 'steam', 'google play' 등의 키워드를 입력 시 하단 링크 태그 영역으로 자동 분리되던 현상을 유저 요청으로 제거. 이제 모든 텍스트 기반 태그는 한 영역(위쪽)에 통일되어 노출됨.
- **포트폴리오 카드 대표 이미지 변경 기능**: 포트폴리오 갤러리 카드 썸네일에 '업로드' 아이콘 버튼을 추가하여, 관리자가 해당 카드의 대표 이미지를 직접 업로드하고 변경할 수 있도록 구현.
- **상세보기(ProjectDetail) 미디어 탭 시스템 개편 및 레이아웃 재배치**: 첨부된 시안에 맞춰 전체 레이아웃을 완전 개편.
  - **최상단 텍스트 정보**: 프로젝트 카테고리, 일반 태그, 제목, 설명, 링크 태그(출시 태그)가 가장 상단에 직관적으로 노출.
  - **미디어 뷰어 (중앙)**: 대표이미지, 영상, PDF가 뜨는 뷰어 영역.
  - **작업 문서 (PDF)**: 
    - 좌측 문서 리스트 + 우측 뷰어의 2단 분할 레이아웃으로 여러 PDF를 등록/열람 지원. 기본값으로 목록창이 열려 있습니다.
